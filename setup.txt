=======================================
 ChromaDB + Gemini Vector DB Setup Guide
=======================================

This file contains the complete setup instructions
to run the ChromaDB vector database project using
Google Gemini embeddings.



---------------------------------------
1. Clone the Repository
---------------------------------------
git clone https://github.com/yourusername/chromadb-gemini-demo.git
cd chromadb-gemini-demo




---------------------------------------
2. Create a Virtual Environment
---------------------------------------
python -m venv venv
# Activate on Windows:
venv\Scripts\activate
# Activate on Linux/Mac:
source venv/bin/activate




---------------------------------------
3. Install Dependencies
---------------------------------------
pip install -r requirements.txt




---------------------------------------
4. Create .env File
---------------------------------------
In the project root, create a file named ".env" with the following:

CHROMA_API_KEY=***
CHROMA_TENANT_ID=***
CHROMA_DATABASE=***
GOOGLE_API_KEY=***




---------------------------------------
5. Verify Requirements
---------------------------------------
Ensure that requirements.txt contains:
chromadb
google-generativeai
python-dotenv

If not, add them manually.



---------------------------------------
6. Prepare Input File
---------------------------------------
Create a text file named "data.txt" in the root directory.
Add any sample text you want to embed into ChromaDB.



---------------------------------------
7. Run the Script
---------------------------------------
python main.py
